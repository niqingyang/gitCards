# wp-gitcard

本项目 fork 于 gitCards，按照自己喜欢的样式进行了一些细微的修改

-----

## 如何自定义样式

scss源文件位于src文件夹中，如有需要可自行修改。

运行`npm start`会监听src文件夹的任何修改操作并实时生成开发环境的代码

运行`npm run build`会生成用于生产环境的代码
